# Personal Portfolio Website

A responsive and modern portfolio website built using HTML, CSS, and Bootstrap. This project showcases my skills, projects, resume, and contact information in a clean and user-friendly interface.

## Features

- Responsive design for all devices
- Modern and clean UI
- Smooth navigation
- Project showcase section
- Skills and about section
- Contact form/social links
- Built with Bootstrap components

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
